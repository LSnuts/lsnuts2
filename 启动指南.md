# lsnuts2 项目启动指南

## 📋 项目概述

本项目是一个云端社区平台，包含：
- **前端**：Vue 3 + Element Plus（部署在 Cloudflare Pages）
- **后端**：Flask + SocketIO（运行在本地）
- **数据库**：PostgreSQL（本地数据库）
- **隧道**：Cloudflare Tunnel（连接本地服务到云端）

---

## 🚀 快速启动

### 方法一：使用启动脚本（推荐）

双击运行：
```
D:\项目\lsnuts2\start.ps1
```

脚本会自动：
1. ✅ 启动 PostgreSQL 数据库
2. ✅ 启动后端服务（端口 5000）
3. ✅ 启动 Cloudflare Tunnel

### 方法二：手动启动

如果脚本无法运行，按以下步骤操作：

#### 1. 启动 PostgreSQL 数据库
```powershell
# 以管理员身份运行 PowerShell
Start-Service postgresql-x64-18
```

#### 2. 启动后端服务
```powershell
cd D:\项目\lsnuts2\backend
D:\miniconda3\python.exe app.py
```

#### 3. 启动 Cloudflare Tunnel
```powershell
cd D:\项目\lsnuts2
.\cloudflared.exe tunnel --config cloudflare-tunnel.yml run
```

## ⚠️ 常见问题

### Q1: 启动脚本提示"无法执行脚本"
**原因**：PowerShell 执行策略限制

**解决**：
```powershell
# 以管理员身份运行 PowerShell
Set-ExecutionPolicy RemoteSigned
# 按 Y 确认
```

### Q2: 网站白屏/无法登录
**原因**：后端服务或数据库未启动

**解决**：
1. 检查 PostgreSQL 服务是否运行
2. 检查后端服务是否在端口 5000 监听
3. 检查 Cloudflare Tunnel 是否运行

### Q3: 手机端访问问题
**解决**：
1. 确保手机网络正常
2. 清除浏览器缓存（强制刷新）
3. 尝试切换网络（WiFi/流量）

### Q4: Cloudflare Tunnel 连接失败
**检查**：
```powershell
# 查看 Tunnel 状态
.\cloudflared.exe tunnel info lsnuts-tunnel
```
